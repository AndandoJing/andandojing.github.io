Learn Python the hard way-Ex33 while循环

2019-06-16

while-loop（while 循环）会一直执行代码，直到对应的布尔表达式为False时才会停下来。

如果某一行是以冒号colon结尾，那就意味着接下来的内容是一个新的代码片段，新的代码片段是需要被缩进的。

因为while如果不给一个结束点会一直循环，所以：

  • 尽量少用while-loop，大部分时候用for-loop就行；
  • 检查while语句，确定布尔表达式最后会变成False；
  • 如果不确定，就在while-loop的结尾打印出要测试的值。

ex33.py的代码：

i = 0
numbers = []
while i < 6:
    print("At the top i is %d" % i)
    numbers.append(i)
    i = i + 1
    print("Numbers now: ", numbers)
    print("At the bottom i is %d" % i)
print("The numbers:")
for num in numbers:
    print(num)

加分习题：

  • 将while循环改成一个函数，将测试条件（i < 6）中的6换成一个变量；
  • 使用这个函数重写脚本，并用不同的数字测试；
  • 为函数添加另一个参数，这个参数用来定义i = i + 1的加值+ 1，这样就可以任意加值；
  • 再使用该函数重写一遍这个脚本，看看效果；
  • 使用for-loop和range把脚本再写一遍，看是否还需要中间的加值操作。

加分代码：
练习1:

import random
i = 0
numbers = []
plus = random.randint(1,5)
replace_num = plus + 1
print("Plus number is %d" % plus)
print("Replace number is %d" % replace_num)
for i in range(0, 10):
    if i < replace_num:
        print("At the top i is %d" % i)
        numbers.append(i)
        i += plus
        print("Numbers now: ", numbers)
        print("At the bottom i is %d" % i)
    else:
        print("i: %d is greater or equal to replace number." % i)
print("The numbers:")
for num in numbers:
    print(num)

练习2:

def list(i, plus):
    numbers = []
    start = 0
    while start < i:
        print("The last number is %d" % start)
        numbers.append(start)
        start += plus
        print("Numbers now:", numbers)
        print("The next number is %d" % start)
i = int(input("Type a number here:"))
plus = int(input("Type the plus number here:"))
list(i, plus)

for-loop和while-loop的区别：
for-loop只能对一些东西的集合进行循环，while-loop可以对任何对象进行驯化。while-loop更难使用，一般任务用for-loop更容易。
