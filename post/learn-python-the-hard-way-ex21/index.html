Learn Python the hard way-ex21

2019-06-07

使用 = 和return将变量设置为“一个函数的值”。

ex21.py的代码：
def add(a, b):
    print("ADDING %d + %d" % (a, b))
    return a + b
def subtract(a, b):
    print("SUBTRACTING %d - %d" % (a, b))
    return a - b
def multiply(a, b):
    print("MULTIPLYING %d * %d" % (a, b))
    return a * b
def divide(a, b):
    print("DIVIDING %d / %d" % (a, b))
    return a / b
print("Let's do some math with just functions!")
age = add(30, 5)
height = subtract(78, 4)
weight = multiply(90, 2)
iq = divide(100, 2)
print("Age: %d, Height: %d, Weight: %d, IQ: %d" % (age, height, weight, iq))
#A puzzle for the extra credit, type it in anyway.
print("Here is a puzzle.")
what = add(age, subtract(height, multiply(weight, divide(iq, 2))))
print("That becomes: ", what, "Can you do it by hand?")

创建函数add, subtract, multiply, divide，函数的最后一行是return a + b（举例），实现的功能是：

  1. 调用函数时使用参数a和b
  2. 打印函数的功能
  3. 告诉python让它做回传的动作：将a + b的值返回（return）
  4. python将两个数字相加，当函数结束时，将a + b的结果赋予一个变量
    用input()输入自定义值的话,in(input())是输入整数值，float(input())可以输入浮点数。
