Learn Python the hard way-Ex30 Else和If

2019-06-15

python的规则里，只要一行已冒号colon结尾，接下来的内容就应该有缩进。

ex30.py的代码部分：

people = 30
cars = 40
buses = 15
if cars > people:
    print("We should take the cars.")
elif cars < people:
    print("We should not take the cars.")
else:
    print("We can't decide.")
if buses > cars:
    print("That's too many buses.")
elif buses < cars:
    print("Maybe we could take the buses.")
else:
    print("We still can't decide.")
if people > buses:
    print("Alright, let's just take the buses.")
else:
    print(“F.ine, let's stay home then.")

练习用多写一些复杂的布尔表达式：
cars > people and buses < cars

⚠️注意
如果多个elif区块都是True，Python只会运行它碰到的第一个True的区块。
