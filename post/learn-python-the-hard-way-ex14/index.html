Learn Python the hard way-ex14

· 2019-05-28 · # python 
from sys import argv
script, user_name = argv
prompt = '> '
print("Hi %s, I'm the %s script." % (user_name, script))
print("I'd like to ask you a few questions.")
print("Do you like me %s?" % user_name)
likes = input(prompt)
print("Where do you live %s?" % user_name)
lives = input(prompt)
print("What kind of computer do you have?")
computer = input(prompt)
print ("""
Alright, so you said %r about liking me.
You live in %r. Not sure where that is.
And you have a %r computer. Nice.
""" % (likes, lives, computer))

这是将用户提示符设置为变量prompt，这样就不用在每次用到input () 时重复输入提示用户的字符了。而且在修改提示符为别的字串时，只需要修改一个位置。 加上加分习题后的修改版本（忘记另存为ex14_1.py了，只能重新复制粘贴）：

from sys import argv
script, user_name = argv
prompt = 'Your answer is> '
print("Hi %s, I'm the %s script." % (user_name, script))
print("I'd like to ask you a few questions.")
print("Do you like me %s?" % user_name)
likes = input(prompt)
print("Where do you live %s?" % user_name)
lives = input(prompt)
print("What kind of computer do you have?")
computer = input(prompt)
print("And you like your girlfriend, right?")
girlfriend = input(prompt)
print ("""
Alright, so you said %r about liking me.
You live in %r. Not sure where that is.
And you have a %r computer. Nice.
You said %r when I asked your about if you do like your girlfriend.
Maybe you should tell her now.
""" % (likes, lives, computer, girlfriend))
print("""
But maybe I can be smarter if I use some more variables.
For example, when I use \\t, it means:
\tI can start here.
And if I use \\t*, it means:
\t*Hi everyone, I can start writing from here.
Isn't it cool?
""")

Ex14_1.py中添加了新的参数变量girlfriend，复习了一遍之前的\=\和\t用法
