Learn Python the hard way-Ex40 模块、类、对象

2019-06-24

python是一种“面向对象编程语言（object-oriented programming language）”，也即python中有一种叫做class的结构，通过它可以用一种特殊的方式构造软件。（理论上来说）使用class（类）可以让程序架构更加整齐，使用起来更干净。

模组和字典差不多

举例：

mystuff = {‘apple’: “I AM APPLES!”}
pirnt(mystify[‘apple’])

这样是从字典中取除key为’apple’的对应值。

“从Y获取X”的概念是字典的基本逻辑。

模组module：

  1. 模组是包含函数和变量的python文件
  2. 可以import这个文件
  3. 可以使用’.’操作符访问到模组中的函数和变量 举例：
#this goes in mystuff.py
def apple():
    print(“I AM APPLES!”)

这是在mystuff.py的模组中放了一个叫apple的函数。

现在来import调用这个模块并访问到apple函数：

import mystuff
mystuff.apple()

还可以放一个叫做tangerine的变量到模块里：

def apple():
    print(“I AM APPLES!”)
#this is just a variable
tangerine = “Living reflection of a dream”

添加变量后还可以访问到这个变量：

import mystuff
mystuff.apple()
print(mystify.tangerine)

回到字典概念，对比模组和字典的使用方法和语法：

mystuff[‘apple’] #get apple from dict
mystuff.apple() #get apple from the module
mystuff.tangerine #same thing, it’s just a variable

也就是说，python 里有一个通用的模式：

  1. 拿一个类似key=value风格的数据容器
  2. 通过key的名称获取其中的value 对于字典来说，key是一个字符串，获得值的语法是[key]。对于模组来说，key是函数或者变量的名称，而语法是.key。

类和模组差不多 也可以把模组当作一种特殊的字典，用来储存一些python代码，通过’.’操作符访问这些代码。python还有另一种代码结构来实现类似目的，也就是类（class），通过类，可以把一组函数和数据放到一个容器中，并用’.’操作符访问到它们。 举例：

class MyStuff(object):
    def __init__(self):
        self.tangerine = “And now a thousand years between”
    def apple(self):
        print(“I AM CLASSY APPLES!”)

这里是用创建mystuff模块的方法来创建一个类。

这段代码大概就是模拟了一个叫MyStuff的迷你模块，里面有一个叫apple()的函数，还有一个叫__init__()的函数（暂时不懂什么意思），同时设置了tangerine变量时用了self.tangerine的语法。

类与模组使用上的区别： 类可以重复创建很多出来，哪怕一次一百万个，也不会互相干涉（相对独立）。但模组，一次import后，整个程序里就只有这一个内容，只有把模组内部探索得更深才行。

对象object是什么？ 对象相当于迷你版的import 如果说类和迷你模组差不多，那么对于类来说，也有一个类似import的概念，这个概念叫做“实例（instance）”，意思就是“创建”。当讲一个类“实例化”以后，就会得到一个对象（object）。 举例：

thing = MyStuff() #“实例化”操作，类似调用函数
thing.apple()
print(thing.tangerine)

实现实例化的方法，就是像调用函数一样调用一个类。

解释：

  1. python看到了MyStuff()并且知道了它时定义过的一个类；
  2. python创建了一个空的对象，里面包含了在类中用def创建的所有函数；
  3. python回去检查你是不是在里面创建了一个__init__魔法函数，如果有创建，就回调用这个函数，从而对你的空对象实现初始化；
  4. 在MyStuff中的__init__函数里，我们有一个多余的函数叫做self，这就是python为我们创建的空对象，而我可以对它进行类似模组、字典等操作，为它设置一些变量进去；
  5. 这里把self.tangerine设成一段歌词，这样就初始化了该对象；
  6. 最后python将这个新建的对象赋给一个叫thing的变量，以供以后使用。

⚠️注意： 类和对象和模组是完全不同的东西。 从东西里获取东西的三种办法：

#dict style
mystuff[‘apples’]

#module style
mystuff.apples()
print(mystuff.tangerine)

#class style
thing = MyStuff()
thing.apples()
print(thing.tangerine)

为什么创建init或者别的类函数时需要多加一个self变量？ 如果不加self，cheese = ‘Frank’这样的代码意义就不明确了，它指的既可能是实例的cheese属性，或者一个叫做cheese的局部变量。有了self.cheese = ‘Frank’就清楚地知道这指的是实例的属性self.cheese.

class类的详细解释： class包括：

  1. 类的属性：类中涉及的变量
  2. 类的方法；类中的函数

init()是什么？

  1. xx()这类的函数（方法）是声明该属性为私有，不能再类的外部被使用或者直接访问。
  2. init()支持带参数的类的初始化，也可以声明该类的属性。
  3. init()的第一个参数必须是self（self为习惯用法，可以改成别的名字），后续参数可以自由指定，和 定义函数没有区别。

函数定义 在类中，def可以为类定义一个函数（方法），与一般函数定义不同，类方法必须包含参数self，且为第一个参数。

self参数含义 在类的代码（函数）中，需要访问当前的实例中的变量和函数。 举例： 访问Instance中的对应变量property： Instance.ProperyNam，读取之前的值和写入新的值 调用对应函数function： Instance.function()，即执行对应动作 这里的Instance本身就是self
