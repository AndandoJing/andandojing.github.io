Learn Python the hard way-Ex38 列表的操作

2019-06-23

列表追加append操作，做一个复习

举例：mystuff.append(‘hello’)的工作原理：

  1. python看到使用到了mystuff，于是去找到这个变量；
  2. 找到mystuff后，开始处理句点.period这个操作符，开始查看mystuff内部的一些变量。由于mystuff是一个列表，python知道mystuff支持一些函数；
  3. 处理append。python会将append和mystuff支持的所有函数的名称一一对比，如果确实其中有一个叫append的函数，那么python就会去使用这个函数；
  4. python看到了括号( )parenthesis并且意识到这是一个函数。到这里，就正常调用这个函数，不过这里的函数还要多一个参数才行；
  5. 这个额外的参数就是mystuff。可能有点奇怪，但是可以直接记住使用。真正发生的是append(mystuff, ‘hello’)，不过我们看到的是mystuff.append(‘hello’)。

举例：

$ python Python 2.6.5 (r265:79063, Apr 16 2010, 13:57:41) [GCC 4.4.3] on linux2 Type "help", "copyright", "credits" or "license" for more information. >>> class Thing(object): ... def test(hi):
... print "hi" ...
>>> a = Thing() >>> a.test("hello" ) Traceback (most recent call last):
File "<stdin>" , line 1, in <module> TypeError: test() takes exactly 1 argument (2 given)
>>>

这里的错误显示的是
test() takes exactly 1 argument (2 given)
也就是说test()只可以接受两个参数，但实际上给了一个。意味着python把a.test(“hello”)改成了test(a, “hello”)，有人弄错了，没有添加a参数。

ex38.py代码：

ten_things = "Apples Oranges Crows Telephone Light Sugar"
print("Wait there's not 10 things in that list, let's fix that.")
stuff = ten_things.split(' ')
more_stuff = ["Day", "Night", "Song", "Frisbee", "Corn", "Banana", "Girl", "Boy"]
while len(stuff) != 10:
    next_one = more_stuff.pop()
    print("Adding: ", next_one)
    stuff.append(next_one)
    print("There's %d items now." % len(stuff))
print("There we go: ", stuff)
print("Let's do some things with stuff.")
print(stuff[1])
print(stuff[-1]) #whoa! fancy
print(stuff.pop())
print(''.join(stuff)) # what? cool!
print('#'.join(stuff[3:5]))#super stellar!

针对代码的部分疑惑及解释：
pop()函数
用于移除列表中的一个元素（默认最后一个元素），并且返回该元素的值。
**语法：**list.pop([index = -1])
参数：
obj——可选参数，要移除列表元素的索引值，不能超过列表总长度，默认为index= -1，删除最后一个列表值。
返回值：
该方法返回从列表中移除的元素对象
实例：

#!/user/bin/python3
#coding=utf-8
list1 = [‘Google’, ‘Runoff’, ’Taobao’]
list_pop = list1.pop(1)
print(“删除的项为：”, list_pop)
print(“列表现在为：”, list1)
=>
删除的项为：Runoob
列表现在为：[‘Google’, ’Taobao’]

len()函数：
len()方法返回对象（字符、列表、元组等）长度或项目个数。
**语法：**len(s)
**参数：**s--对象
**返回值：**返回对象长度
实例：

>>>str = “runoob”
>>>len(str) #字符串长度
6
>>> l = [1, 2, 3, 4, 5]
>>> len(l)  #列表元素个数
5

加分习题：
1. 将每个被调用函数按照本课方式翻译成python实际执行的动作。
例如：’ ‘.join(things)其实是join(‘ ‘, things)。
stuff = ten_things.split(' ')实际执行为stuff = split(ten_things, ‘ ‘)
next_one = more_stuff.pop()实际执行为next_one = pop(more_stuff)
stuff.append(next_one)实际执行为append(stuff, next_one)
stuff.pop()实际执行为pop(stuff)

2. 将这两种方式翻译成自然语言。
例如：’ ‘.join(things)可以翻译成“用’ ‘连接(join) things”，而join(‘ ‘, things)的意思是“为’ ‘和things调用join函数”.其实是同一件事情。

3. 面向对象编程object oriented programming（简称OOP）。

4. python中的class是什么。
python中定义类是通过class关键字，例如：
class Student(object):
pass
这样定义一个储存学生信息的类。
class后面紧接着类名，即Student，类名通常是大写开头的单词，紧接着是(object)，表示该类是从哪个类继承下来的。通常，如果没有合适的继承类，就是用object类，这事所有类最终都会继承的类。
定义好了Student类，就可以根据Student类创建出Student的实例，创建实例是通过类名+()实现的：
>>>bart = Student()
>>>bart
<main.Student object at 0x10a67a590>
>>>Student
<class ‘main.Student’>

        可以看出，变量bart指向的就是一个Student的实例，后面的0x10a67a590是内存地址，每个object的地址都不一样，而Student本身则是一个类。
        可以自由地给一个实例变量绑定属性，比如，给实例bart绑定一个name属性：
        >>>bart.name = ‘Bart Simpson’
        >>>bart.name
        ‘Bart Simpson’

5. dir(something)和something的class有什么关系。
dir()函数返回的是list，用来寻找一个对象的所有属性。
dir()函数不带参数时，返回当前范围内的变量、方法和定义的类型列表；带参数时，返回参数的属性、方法列表。如果参数包含方法__dir__()，该方法将被调用。如果参数不包含__dir__()，该方法将最大限度地收集参数信息。

6. 了解OOP（object oriented programming）和函数编程functional programming。
函数编程（functional programming）
或称函数程序设计，是一种编程范型，将电脑运算视为数学上的函数计算，并且避免使用程序状态以及易变对象。函数编程语言最重要的基础是λ演算（lambda calculus）。而且λ演算的函数可以接受函数当作输入（引数）和输出（传出值）。
面向对象程序设计（object-oriented programming，OOP）
是具有对象概念的程序编程范型，也是一种程序开发方法。可能包含数据、属性、代码与方法。对象则指的是类的实例。它将对象作为程序的基本单元，将程序和数据封装其中，以提高软件的重用性、灵活性和拓展性，对象里的程序可以访问及经常修改对象相关联的数据。在面向对象程序编程里，计算机程序会被设计成彼此相关的对象。
对象与对象之间的关系是面向对象编程首要考虑的问题，而在函数编程中，所有的数据都是不可变的，不同的函数之间通过数据流来交换信息，函数作为FP中的一等公民，享有和数据一样的地位，可以作为参数传递给下一个函数，同时也可以作为返回值。

stuff[3:5]实现了什么功能？
列表切片动作，会从stuff列表的第3个元素取值取到第5个元素（但不包括第5个元素）。这和range(3,5)是一样的。

为什么join(‘ ‘, stuff)不灵？
join的文档写的有问题，其实它是要插入的字符串的一个方法函数，函数的参数是你要连接的字符串构成的数组，所以应该写成’ ‘.join(stuff)。
