Learn Python the hard way-ex18

2019-06-01

函数的三个作用：

  1. 给代码片段命名，就和“变量”给字符串和数字命名一样；
  2. 可以接受参数，就像脚本接受argv一样；
  3. 通过使用#1和#2，可以创建“微型脚本”或者“小命令”

这个是ex18.py的代码部分：

#this one is like your scripts with argv
def print_two (*args):
    arg1, arg2 = args
    print("arg1: %r, arg2: %r" % (arg1, arg2))
#ok, that *args is actually pointless, we can just do this
def print_two_again(arg1,arg2):
    print("arg1: %r, arg2: %r" % (arg1, arg2))
#this just takes one argument
def print_one(arg1):
    print("arg1: %r" % arg1)
#this one takes no arguments
def print_none():
    print("I got nothin.")

print_two("Zed","Shaw")
print_two_again("Zed","Shaw")
print_one("First!")
print_none()

这里拆解ex18.py中第一个print_two的每一部分：

  1. 首先是用def命令创建一个函数，def是定义（define）的意思。
  2. def是函数的名称，在这里它的名称是”print_two”，名称可以随便更改，但最好能够体现函数的功能。
  3. 告诉函数需要*args（asterisk args），折合脚本的argv很像，参数必须在()中才能正常工作。
  4. 用冒号结束本行，开始下一行缩进。
  5. 冒号以下，使用4个空格缩进的行都是属于print_two这个函数的内容。其中第一行的作用是将参数解包。这也和argv差不多
  6. 把解包后的参数都打印出来。

这里的print_two并不是最精简的创建函数的办法，python中可以跳过解包过程，直接使用()里面的名称作为变量名。也就是print_two_again表现出来的。

print_one展示函数如何接受单个参数。 print_none展示函数可以不接受任何参数。

args的的意思： 告诉python让它把函数的所有参数都接受进来，然后放到名字叫args的列表中，和argv差不多，但这个args用在函数上。没特殊情况一般不会用到这个东西。

函数注意事项：

  1. 函数定义以def开始
  2. 函数名称用字符和下划线_组成，不可以用数字开始
  3. 函数名称后紧跟()
  4. ()中包含参数，多个参数时用逗号,隔开
  5. 参数名称不可重复（不能使用重复的参数名）
  6. (参数)之后跟着冒号:
  7. 下一行需要缩进，4个空格的缩进（indent），但凡是后面这样缩进的都在这个函数里
  8. 函数结束的位置要取消缩进（dedent） 例子： def first(variable1, variable2):

运行函数时（use或者call）要检查：

  1. 调运函数时是否使用了函数的名称？
  2. 函数名称是否紧跟着()？
  3. ()内有无参数？多个参数是否以逗号隔开？
  4. 函数是否以)结尾？ 也就是是否符合上面的书写规范。

运行函数（run）=调用函数（call）=使用函数（use） 说法不同而已
