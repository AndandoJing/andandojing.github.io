from sys import argv #导入argv模组
script, input_file = argv #设定参数
print("Opening the file...")
target = open(input_file, 'w') #打开文件
print("Truncate the file...") #清空文件内容
target.truncate()
print("Rewrite the file...") #提示用户重新输入文件内容
line1 = input("line1:")
line2 = input("line2:")
line3 = input("line3:")
target.write(line1+'\n'+line2+'\n'+line3) #确认重写内容
target.close() #关闭文件，将内容写入硬盘
def print_all(f):   #定义打印文件函数
    print(f.read())
def rewind(f): #设置开始的偏移量，即从偏移位置开始运行
    f.seek(0)
def print_a_line(line_count, f): #打印行数和该行内容
    print(line_count, f.readline())
current_file = open(input_file) #打开文件，将内容临时赋值给current_file
print("First let's print the whole file:\n")
print_all(current_file)
print("Now let's rewind, kind of like a tape.")
rewind(current_file)
print("Let's print three lines:")
current_line = 1
print_a_line(current_line, current_file)
current_line = current_line + 1
print_a_line(current_line, current_file)
current_line = current_line + 1
print_a_line(current_line, current_file)

file中的seek()是做什么的？
seek()用于移动文件读取指针到指定位置
seek()的语法：
fileObject.seek(offset[,whence])
offset：开始的偏移量，也就是代表需要移动偏移的字节数
whence：可选，默认值为0。给offset参数一个定义，表示要从哪个位置开始偏移；0代表从文件开头开始算起，1代表从当前位置开始算起，2代表从文件末尾算起。
返回值：
该函数没有返回值。
seek()函数
处理对象是字节而不是行，所以seek(0)只会转到文件的0 byte，也就是第一个byte，不会把current_line设为0。current_line只是一个独立变量，和文件本身没有关系，只能手动为其增值。

+=这个简写操作符的作用：
a += b 是 a = a + b 的简写，也就是将a + b重新赋值给a，一般用于计数。比如这个代码中的current_line = current_line + 1可以写成current_line += 1

print_all (f)这里的f是什么？
f是变量，在这里指一个文件。python里的文件读取和DVD或磁带类似，有一个用来读取数据的“磁头”。
运行f.seek()可以回到文件的开始，f.readline()会读取文件的一行，然后将“磁头”移动到\n后面。
readline()函数返回的内容中本来就包括\n（一个空行），print()在打印时也会添加一个\n，这样就会出现两个空行。如果要减掉空行，可以在print()语句结尾加一个逗号，这样print就不会把它自己的\n打印出来了。

readline()
里面的代码会扫描文件的每一个字节，直到找到一个\n为止，然后停止读取文件，并且返回此前的文件内容。文件f会记录每次调用readline()后的读取位置，所以下次调用时可以直接读取接下来的一行。
