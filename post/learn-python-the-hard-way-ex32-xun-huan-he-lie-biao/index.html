Learn Python the hard way-Ex32 循环和列表

2019-06-16

for-loop（for循环） 在开始for循环之前，需要在某个位置存放循环的结果，最好的办法是使用列表list。 for-loop可以使用未定义的变量，因为循环开始时变量就被定义了，每次循环时会被重新定义一次。

创建列表： 举例： hairs = ['brown', 'blond', 'red'] eyes = ['brown', 'blue', 'green'] weights = [1, 2, 3, 4]

列表构成： 列表名 = [‘元素1’, ‘元素2’, ‘元素3’]

ex32.py的代码：

the_count = [1, 2, 3, 4, 5]
fruits = ['apples', 'oranges', 'pears', 'apricots']
change = [1, 'pennies', 2, 'dimes', 3, 'quarters']
#this first kind of for-loop goes through a list
for number in the_count:
    print("This is count %d" % number)
#same as above
for fruit in fruits:
    print("A fruit of type: %s" % fruit)
#also we can go through mixed lists too
#notice we have to use %r since we don't know what's in ti
for i in change:
    print("I got %r" % i)
#we can also build lists, first start with an empty one
elements = []
#then use the range function to do 0 to 5 counts
for i in range(0, 6):
    print("Adding %d to the list." % i)
    # append is a function that lists understand
    elements.append(i)
#now we can print them out too
for i in elements:
    print("Element was: %d" % i)

range函数的用法 range(start, stop[, step])

  • start：计数从start开始。默认是从0开始。例如：range(5)等价于range(0, 5) => [0, 1, 2, 3, 4]；
  • stop：计数到stop结束，但不包括stop。例如：range(0, 5)是[1, 2, 3, 4]没有5；
  • step：步长，默认是1.例如：range(0, 30, 5)等价于range(0, 5, 10, 15, 20, 25) 实例： range在for中使用，循环出runoob的每个字母：

Python3 中range()函数返回的是一个可迭代对象（类型是对象），而不是列表类型，所以打印的时候不会打印列表。 Python3 中list()函数是对象迭代器，可以把range()返回的可迭代对象转为一个列表，返回的变量类型为列表。 Python2 range()函数返回的是列表。

尝试将elements直接赋值为range(0, 6)：

#直接将elements赋值为range(0. 6)且不使用for循环
elements = range(0, 6)

python中的append()用法： list.append(obj) obj：添加到列表末尾的对象 返回值：该方法无返回值，但是会修改原来的列表 作用：在列表里添加新的项 举例：

aList = [123, ‘xyz’, ‘zara’, ‘abc’];
aList.append( 2009 );
print(“Updated List: “, aList);
=>
Updated List: [123, ‘xyz’, ‘zara’, ‘abc’, 2009]

append()和extend()的区别： extend将它的参数视为list，extende的行为是把两个list接到一起。 append将它的参数视为element，作为一个整体添加上去的。

二维列表： 在列表中包含列表。 举例： list1 = [[1, 2], [3, 4], [5, 6]] 访问元素：list1[0] =>[1, 2]

for i in range(1, 3)中range(1, 3)只包括首项到倒数第二项。
