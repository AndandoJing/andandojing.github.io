seek() 移动文件读取指针到指定位置
seek()：fileObject.seek(offset[,whence])
offset: 开始偏移量，也就是代表需要移动偏移的字节数
whence：可选，默认值为0.给offset参数一个定义，表示要从哪个位置开始偏移；0代表从文件开头开始算起，1代表从当前位置开始算起，2代表从文件末尾算起。
返回值：该函数没有返回值
处理对象是字节而不是行。所以seek(0)只会转到文件的0 byte，也就是第一个byte，不会影响行数。		 f.seek()可以回到文件的开始

open()的打开方式：

模式 符号 意义 注意点 举例
读取模式read r 只读
r+ 读写，不创建 可读可写，若文件不存在，报错。r+会对之前写入的东西进行覆盖写 fd = open(“1.txt”, ‘r+’)
fd.write(‘456’)
写入模式write w 新建只写，不能读出 w和w+这两个都会将文件内容清零
w+ 新建写读，可读写 可读可写，若文件不存在，创建 fd = open(“1.txt”, ‘w+’)
fd.write(‘123’)
追加模式 append a 附加写的方式打开，不可读
a+ 附加读写的方式打开 fd = open(“1.txt”, ‘a+’)
fd.write(‘789’)
=>
456789

+：修饰符，文件将以同时读写的方式打开
w和a：不可读
a，a+，w，w+：若不存在会创建新文件
open(filename)的默认打开方式是’r’。open本身对文件的写入操作态度是安全第一，只有特定后才会进入写入操作。
