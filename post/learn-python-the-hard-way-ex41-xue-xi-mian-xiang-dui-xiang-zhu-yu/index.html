面向对象术语词汇表

  • class：告诉python创建新类型的东西。
  • object：两个意思：最基本类型的东西，或者某个东西的实例。
  • instance：当让python创建一个class的时候，我们所得到的东西。
  • def：在类里面定义函数的方法。
  • self：class里面的函数会用到这个参数，self指代被访问的object或者instance的一个变量。
  • inheritance：指一个class可以继承另一个类的特性，和父子关系类似。
  • composition：指一个class可以将别的类作为它的部件构建起来，有点像车子和车轮的关系。
  • attribute：class的一个属性，它来自于composition，而且通常是一个变量。
  • is-a：用来描述集成关系，例如Salmon is-a Fish。
  • has-a：用来描述某个东西时另外一些东西组成的，或者某个东西有某个特征。例如Salmon has-a mouth。

描述方式：

  • class X(Y)
    "Make a class named X that is-a Y." #创建一个叫X的类，它从属于Y。
  • class X(object): def init(self, J)
    "class X has-a init that takes self and J parameters." #class X 有一个__init__接收self和J作为参数。
  • class X(object): def M(self, J)
    “class X has-a function named M that takes self and J parameter.” #class X有一个函数名称为M，它接收self和J作为参数。
  • foo = X()
    “Set foo to an instance of class X.” #将foo设为class X的一个实例。
  • foo.M(J)
    “From foo get the M function, and call it with parameters self, J.” #从foo中找到M函数，并使用self和J参数调用它。
  • foo.K = Q
    “From foo get the K attribute and set it to Q.” #从foo中获取K属性，并将其设为Q。

每一条中的X、Y、M、J、K、Q、foo，都可以将它们当作空白点来对待。

ex41.py的代码（用于阅读练习）：

import random  #导入random随机模组
from urllib.request import urlopen #python3中使用urllib.request，导入urlopen模组
import sys #导入sys

WORD_URL = "http://learncodethehardway.org/words.txt" #确认要导入的文本链接，网页文本中都是单独成行的单词
WORDS = [] #将文本链接中的文本取下来放入一个列表，列表名叫WORDS

PHRASES = {  #PHRASES是整个字典的名字
       "class %%%(%%%):":  #编写脚本时前为key，后为value，一一对应形成字典dict
         "Make a class named %%% that is-a %%%.",
       "class %%%(object):\n\tdef __init__(self, ***)":
         "class %%% has-a __init__ that takes self and *** params.",
       "class %%%(object):\n\tdef ***(self, @@@)":
         "class %%% has-a function *** that takes self and @@@ params.",
       "*** = %%%()":
         "Set *** to an instance of class %%%.",
       "***.***(@@@)":
         "From *** get the *** function, call it with params self, @@@.",
       "***.*** = '***'":
         "From *** get the *** attribute and set it to '***'."
}

#do they want to drill phrases first
if len(sys.argv) == 2 and sys.argv[1] == "english":#判断参数长度是否为2，第二个参数是否为“english”
  PHRASE_FIRST = True #字典的第一个元素是True还是False
else:
  PHRASE_FIRST = False
	
#load up the words from the website
for word in urlopen(WORD_URL).readlines(): #一行一行从网页中读取数据
  WORDS.append(str(word.strip(),encoding="utf-8"))  #删除每一行开始和结尾的空格，只留下单词并加入到words列表中
#函数str.strip([chars])用来移除字符串头尾制定的字符，默认空格，需要移除的字符直接写在[]内
#函数定义部分，操作部分直接跳到try部分先看
def convert(snippet, phrase):  #定义函数convert，包括两个参数snippet和phrase
  class_names = [w.capitalize() for w in   #函数str.capitalize()用来返回一个首字母大写，其他字母小写的字符串
                 random.sample(WORDS, snippet.count("%%%"))]  #从WORDS列表中随机选一个单词，将其首字母大写，然后赋值给class_names。这里假设为“Actor”
  other_names = random.sample(WORDS, snippet.count("***"))   #从WORDS列表中随机取出一个单词赋值给other_names。这里假设为“dinner”
  results = []            #函数random.sample(sequence, k)，sequence是列表（需先被定义），k是需要随机取的数，需为整数
  param_names = []        #函数str.count(sub, start = 0, end = len(str))，返回子字符串在字符串中出现的次数
  for i in range(0, snippet.count("@@@")):  #循环一次
    param_count = random.randint(1,3)      #random.randint(a, b)用于生成一个制定范围内的整数。a <= n <= b.假设随机到了2
    param_names.append(','.join(
      random.sample(WORDS, param_count)))  #从WORDS中随机取出2个单词，用逗号和空格连接起来，放在param_names中，假设为“cook，donkey”
			
  for sentence in snippet, phrase:  #循环两次，只说第一次
    result = sentence[:]  #浅拷贝，result = "class%%%(objet):\n\tdef***(self, @@@)"
		
    #fake class name
    for word in class_names:
      result = result.replace("%%%", word, 1)   #result = "class Actor(object):\n\tdef ***(self, @@@)"
    #fake other name
    for word in other_names:
      result = result.replace("***", word, 1)  #result = "class Actor(object):\n\tdef dinner(self, @@@)"
			
    #fake parameter lists
    for word in param_names:
      result = result.replace("@@@", word, 1)  #result = "class Actor(object):\n\tdef dinner(self, cook, donkey)"
			
    results.append(result)   #添加到results序列中
		
  return results
#results = ["class Actor(object):\n\tdef dinner(self, cook, donkey)", "class Actor has-a function named dinner that takes self and cook, donkey parameters."]
#keep going until they hit CTRL-D
try:
  while True:
    snippets = list(PHRASES.keys())  #提取PHRASES中的key为snippets组成列表
    random.shuffle(snippets)   #打乱snippets中元素（PHRASES中key）顺序
   
	 for snippet in snippets:   #取一个key来操作，假设取到“class %%%(object):\n\tdef ***(self, @@@)”
      phrase = PHRASES[snippet] #取出value，这里的phrase对应的是value，按照上述假设就会是"class %%% has-a function named *** that takes self and @@@ parameters."
      question, answer = convert(snippet, phrase)  #question对应key（snippet），answer对应value（phrase）
      if PHRASE_FIRST:
        question, answer = answer, question
				
      print(question)
			
      input("> ")
      print(f"ANSWER: {answer}\n\n")
except EOFError:
  print("\nBye")

整个代码的流程如下：

  • 开始；
  • 从网页读取单词写入WORDS形成列表；
  • 定义PHRASES为字典，并设置好字典形式，留出需要被替换的字符位置；
  • 提取PHRASES字典中的key键存入snippets，并打乱顺序；
  • 从snippets中取出一个元素snippet；
  • 取出PHRASES字典中对应该snippet键的值value，为phrase；
  • 进入convert函数。确认phrase（value）中有几个“%%%”，就随机从WORDS（单词列表）中取几个单词放入class_names；
  • 将class_names中的单词首字母大写；
  • phrase中有几个“** *” 就随机从WORDS（单词列表）中取几个单词放入other_names；
  • 检查phrase中是否有“@@@”：
    • 有“@@@”的情况下：
      • 在[1, 3]中取随机数赋值给param_count；
      • 在WORDS（单词列表）中取param_count个单词，将它们用”, “连接起来并写入param_names；
    • 没有“@@@”的情况下：
  • 遍历snippet（key）和phrase（value），赋给sentence；
  • result浅拷贝sentence；
  • 用class_names中的单词替换result中的“%%%”；
  • 用other_names中的单词替换result中的“***”；
  • 用param_names中的单词替换result中的“@@@”;
  • 把result作为一个元素加入results中；
  • snippet和phrase处理完之后，返回results（退出convert函数）；
  • 根据PHRASE_FIRST的值决定处理后snippet（key）和phrase（value）的打印顺序；
    • 如果参数长度为2且第二个参数为english（参数个数包括文件名计为1）：
      • PHRASE_FIRST为True
        • 先打印value，后打印key
    • 否则：
      • PHRASE_FIRST为False
        • 先打印key，后打印value
  • 结束

  • 异常处理（使用try）：
    python中捕捉异常可以使用try/except语句，用来检测try语句块中的错误，从而让except语句捕捉异常信息并处理。
    如果不想再异常发生时结束程序，就可以用try来捕捉。
    语法举例(try…except…else)：
try:
<语句>  #运行别的代码
except<名字>:
<语句>  #如果再try部分引发了’name’异常
except<名字>,<数据>:
<语句>  #如果引发了’name’异常，获得附加的数据
else:
<语句>  #如果没有异常发生

执行原理：

  • try中<语句>执行时出现异常，python跳回try并执行第一个匹配该异常的except子句，异常处理完毕，控制流就通过整个try语句（除非发生新异常）；
  • try中<语句>发生了异常，但没有匹配的except子句，异常会被递交到上层try，或到程序最上层（这样会结束程序，并打印缺省的出错信息）；
  • try中<语句>执行时无异常，python将执行else中<语句>（如果有else的话），然后控制流通过整个try语句。

浅拷贝：
直接赋值、浅拷贝、深度拷贝之间的区别：
直接赋值： 对象的引用
浅拷贝copy： 拷贝父对象，不会拷贝对象的内容的字对象
深拷贝deepcopy： copy模块的deepcopy方法，完全拷贝了父对象及其字对象
浅拷贝举例：

>>> a = {1: [1, 2, 3]}
>>> b = a.copy()
>>> a, b
({1: [1, 2, 3]}, {1: [1, 2, 3]})
>>> a[1].append(4)
>>> a, b
({1: [1, 2, 3, 4]}, {1: [1, 2, 3, 4]})

深度拷贝需要引入copy模块，举例：

>>> import copy
>>> c = copy.deepcopy(a)
>>> a, c
({1: [1, 2, 3, 4]}, {1: [1, 2, 3, 4]})
>>> a[1].append(5)
>>> a, c
({1: [1, 2, 3, 4, 5]}, {1: [1, 2, 3, 4]})

解析：

  1. b = a，赋值引用，a和b指向同一个对象；
  2. b = a.copy()，浅拷贝，a和b是独立对象，但它们的子对象还是指向统一对象（是引用）；
  3. b = copy.deepcopy(a)，深度拷贝，a和b完全拷贝了父对象及其子对象，两者是完全独立的
