Learn Python the hard way-ex19

2019-06-02

函数里的变量和脚本里的变量之间没有连接。

这里是ex19.py的内容：

#定义函数cheese_and_crackers，设定两个变量cheese_count和boxes_of_crackers
def cheese_and_crackers(cheese_count, boxes_of_crackers):
    print("You have %d cheeses!" % cheese_count)
    print("You have %d boxes of crackers!" % boxes_of_crackers)
    print("Man that's enough for a party!")
    print("Get a blanket.\n")
#设定这个函数要执行的操作一共是四个打印步骤
#第一种赋予函数变量的值的方式，直接赋予变量相应的值，两个变量对应20和30的值
print("We can just give the function numbers directly:")
cheese_and_crackers(20, 30)
#第二种赋予函数变量的值的方式，用脚本定义两个变量的值
print("OR, we can use variables from our script:")
amount_of_cheese = 10
amount_of_crackers = 50
#再把脚本定义的变量架到函数变量上。
cheese_and_crackers(amount_of_cheese, amount_of_crackers)
#注意，这种办法需要先定义脚本变量，不然没出现过，后面不能识别
#第三种赋予函数变量的值的方式，可以用运算的方式在函数内赋予。
print("We can even do math inside too:")
cheese_and_crackers(10 + 20, 5 + 6)
#第四种赋予函数变量的值的方式，可以结合脚本变量和运算，在函数内赋予。
print("And we can cobine the two, variables and math:")
cheese_and_crackers(amount_of_cheese + 100,
amount_of_crackers + 1000)

可以看出我们在给函数cheese_and_crackers很多参数，然后在函数里把他们打印出来。函数中可以用变量名，可以在函数中运算，也可以将变量和运算结合。 函数的参数和我们生成变量时用的 = 赋值符类似。如果一个物件可以用 = 将其命名，一般也可以将其作为参数传递给一个函数。 如果要处理用户输入的数字，例如想让用户输入cheese和crackers的数量时，除了input()还要记得用int()把输入的值转为整数。

第二种赋予函数变量的值的方式里，借用了脚本里创建变量的方式。这种方式里，脚本变量是在函数变量之外的，不会改变函数中的变量，它们被传递到函数中之后，函数会为之创建临时版本。函数运行结束后，这些临时变量就会被丢弃了。 因此，在设定这种全局变量的名称时，最好不要用同样的变量名称，会弄混。

函数的参数个数虽然有限（取决于python版本和操作系统），但限值很大，实际应用中，5个参数就不少了。

可以在函数中调用函数。

这里尝试自建新函数，并且用十种方式运行：

#尝试自己编至少一个函数出来，并用10中方法运行这个函数
def grado_de_amor(amor_mio, amor_tuyo, distancia):
    print("If my love is %d degree." % amor_mio)
    print("And your love is %d degree." % amor_tuyo)
    print("When we are %d km apart." % distancia)
    print("Our love degree can be calculated.")
    data = (amor_mio + amor_tuyo)/distancia
    print("It's %d degree." % data)
#第一种赋值方法，直接赋值
print("Primera manera:")
grado_de_amor(49, 51, 1)
#第二种赋值方法，脚本参数变量
print("Segunda manera:")
degree_of_my_love = 40
degree_of_your_love = 50
distance = 2
grado_de_amor(degree_of_my_love, degree_of_your_love, distance)
#第三种赋值方法，做运算
print("Tercera manera:")
grado_de_amor(40 + 10, 50 + 10, 3 + 3)
#第四种赋值方法，脚本参数变量和运算合在一起
print("Cuarta manera:")
grado_de_amor(degree_of_my_love + 60, degree_of_your_love + 50, distance + 10)
#第五种赋值方法，颠倒第四种的顺序
print("Quinta manera:")
grado_de_amor(70 + degree_of_my_love, 1 +
degree_of_your_love, 3.3 + distance)
#第六种赋值方式，脚本参数变量，直接赋值，参数+运算
print("Sexta manera:")
grado_de_amor(degree_of_my_love, 80, distance + 6)
#第七种赋值方式，用户输入数值，整数
print("Séptima manera:")
degree_1 = int(input("my love:"))
degree_2 = int(input("your love:"))
degree_3 = int(input("distance:"))
grado_de_amor(degree_1, degree_2, degree_3)
#第八种赋值方式，用户输入数值上做运算、参数混合，浮点数
print("Octava manera:")
degree_1_8 = float(input("my love_8:"))
degree_2_8 = float(input("your love_8:"))
degree_3_8 = float(input("distance_8:"))
grado_de_amor(degree_1_8, degree_2_8 + 10, degree_3_8 + degree_1_8)
#第九种赋值方式，基础值上增加用户输入值
print("Novena manera:")
degree_1_9 = int(input("my love_9:"))
degree_2_9 = int(input("your love_9:"))
degree_3_9 = int(input("distance_9:"))
grado_de_amor(degree_1_9 + int(input("plus degree:")),
degree_2_9 - int(input("minus degree:")),
degree_3_9 + int(input("puls distance:")))
#第十种复制方式，直接在函数参数写法上写成输入格式，不再使用临时的命名
print("Décima manera:")
grado_de_amor(float(input("最后一次我的:")),
float(input("最后一次你的:")),
float(input("最后一次距离：")))

⚠️注意：因为这个函数里有除法，所以最后一个函数的参数不可以时0，同时因为之前都是用int()做，所以所有的数都是整数形式。最后一个改成了浮点数float形式。
